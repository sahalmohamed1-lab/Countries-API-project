About this project.

This website allows you to explore details about various countries worldwide such as their capital cities, the region they are found and their flag.

Features

    Helps users to search for countries and their details.

    Uses a public API to fetch data and display to the user dynamically.

    Has a responsive layout on all devices.

What you can do

    Search for a specific country and view details such as their flag, capital city and the region they are found in.

    Scroll to view the details of the various countries.

How it works

The website uses REST countries API to fetch data and displays it dynamically to the user.

The project uses HTML, CSS and JavaScript for its structure, styling and adding interactivity by fetching data through an API.
