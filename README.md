### Project title
> Country Finder.

## Project description
> A website that allows a user to search for a specific country and view details such as their flag, their capital city and the region they are found in.

## Installation
1. Clone the repository:
```bash 
  git@github.com:sahalmohamed1-lab/Countries-API-project.git
```
2. Install dependencies:
```bash
  npm install
```

## API used
> REST countries API  https://restcountries.com/
> Data used:
- Country Name
- Country flag
- Capital city
- Region

## Features
- Fetches country data from an API.
- Search for countries in real time.
- Displays country name, flag, capital city and region.
- Responsive design for mobile and other devices by use of CSS grids and media queries.
- Uses HTML, CSS, JavaScript and Jest.
- Unit tests to ensure the rendering of logic and correct handling of country data.
